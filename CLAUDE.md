# Asm198x docs

Documentation repo for the Asm198x assembler family. Part of the `asm198x` org
container; see [`../CLAUDE.md`](../CLAUDE.md) for the org layout and
[`../../CLAUDE.md`](../../CLAUDE.md) for the 198x umbrella.

## Scope

- The declarative **ISA spec format** (the shape consumed by the `isa` crate).
- Per-machine **dialect references** (the source-compatible syntax each backend
  targets — see the assembler's
  [`decisions/syntax-stance.md`](../asm198x/decisions/syntax-stance.md)).
- Eventually a docs site.

## Not here

- Hardware facts — those live in the umbrella primary library
  [`../../reference/`](../../reference/) and syntheses, per
  [`../../decisions/shared-hardware-reference-canon.md`](../../decisions/shared-hardware-reference-canon.md).
- The binding architecture decision — umbrella
  [`../../decisions/asm198x-and-shared-isa-spec.md`](../../decisions/asm198x-and-shared-isa-spec.md).

Empty for now; structure will follow the curriculum/docs conventions used
elsewhere in the family once there's content to organise.
