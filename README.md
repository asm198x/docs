# Asm198x docs

Documentation for the [Asm198x](https://github.com/asm198x) assembler family:
the declarative ISA spec format, per-machine dialect references, and (in time)
the docs site.

Empty for now — content lands as the assembler grows. The authoritative
architecture decision lives in the 198x umbrella
([asm198x-and-shared-isa-spec](https://github.com/…/198x/blob/main/decisions/asm198x-and-shared-isa-spec.md)),
and the assembler's own notes in the
[asm198x](https://github.com/asm198x/asm198x) repo's `decisions/`.
